### [Market Grid](https://github.com/Damian34/market-grid)
A microservices-based market data processing app with event-driven architecture.
- Collects and normalizes market and currency data from external sources
- Includes a separate **gateway** and **OAuth2 authentication** handled by a dedicated service
- Environment runs locally, as well as with docker and is designed for extensibility and future Kubernetes deployment
- **Mixed Stack:** Java 23, Kotlin, Spring Boot, Quarkus, Kafka, Maven, Gradle, and more

### [Notes GraphQL API & Kubernetes Lab](https://github.com/Damian34/notes-graphql-k8s-lab)
- GraphQL CRUD API built with Spring Boot using **Hexagonal Architecture**
- Includes Kubernetes manifests and Helm charts for hands-on deployment experiments
- Extensive notes documenting architecture decisions and K8s deployment workflows

### [Spring Security JWT Auth](https://github.com/Damian34/spring-security-jwt-auth)
- Template project demonstrating and exploring **JWT authentication** with **Spring Security**
- Focus on **proper separation of concerns** and bounded contexts for security class structures

### [CI/CD Dual Pipeline Template](https://github.com/Damian34/ci-cd-dual-pipeline-dev-prod-release-template)
- Automates **semantic versioning** in Java/Maven projects using GitHub Actions
- Supports development and production flows with automatic version bumping
